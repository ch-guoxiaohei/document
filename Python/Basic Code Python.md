### Common Function

- print()
print messages to console
- input()
get user input messages
- len()
get length of input.eg: `len('')=0`,`len('bb')=2`
- str()
convert input to str. eg: `str(29)='29'`
- int()
convert input to int. eg: `int('29')=29`
- float()
convert input to float. eg: `fload('29')=29.0`
